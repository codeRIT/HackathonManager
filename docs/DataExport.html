POSSIBLE_TYPES =

A DataExport is a generated .zip of data from HackathonManager, such as a .zip of resumes & attendee data, or a .zip of the entire database is the form of multiple CSVs.

These should be generated asynchronously with a background job, and then stored as an active storage attachment.

def enqueue!
  raise "Data export has already been queued" unless status == "created_not_queued"

  update_attribute(:queued_at, Time.now)
  GenerateDataExportJob.perform_later(self)
end

def file_basename
  time = created_at.strftime("%r").gsub(":", "-")
  date = created_at.strftime("%F")
  "#{export_type} #{date} #{time}"
end
